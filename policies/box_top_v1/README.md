# Box-Top Policy V1

This directory contains the completed first-policy line.

Use it for:

- Historical reward/config timeline.
- Final report and lessons learned.
- Checkpoint references.
- Baseline metrics for v2 comparisons.

Do not keep new v2 design work here.

Key files:

```text
FINAL_REPORT.md      Full technical report and reconstruction.
PROGRESS_REPORT.md   Branch-by-branch progress notes.
GAIT_PLAN.md         Earlier gait quality plan.
README.md           Original root README from the v1 work.
```

Most v1 runtime artifacts are intentionally not stored here because `logs/` and `outputs/` are ignored. The local machine still has those artifacts unless manually deleted.
